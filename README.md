# Ticket Booking System

The project demonstrates basic seat selection, movie information display, and a booking summary panel.
This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 18.2.1.

## Live Demo
https://ticket-booking-sample.netlify.app/

## Features
- Movie information section
- 12 × 8 seat layout
- Seat selection and deselection
- Booking summary with total calculation
- Simple state management using NgRx

Make sure Node.js (version 18 or above) and Angular CLI are installed.
